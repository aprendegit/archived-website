Hola:

No es un servidor lo que tienes que crear sino un remoto. Ve a SourceControl, y selecciona el nombre del proyecto como si fueses a crear una rama. En el menú secundario que se despliega verás como última opción «Configure [nombre del proyecto]». Al seleccionarla se te despliega una ventana emergente. Selecciona la pestaña «Remote» y abajo a la izquierda haz clic en el icono «+». Selecciona «Add remote…». En la ventana que se te abre introduces la URL del repositorio git el campo Address y el nombre del remote (por ejemplo, origin) en el campo Name.

Creo que es esto lo que estás buscando.
